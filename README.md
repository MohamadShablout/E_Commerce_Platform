# E-Commerce Platform

This is a simple online store website. It is not a real store.
I made this project to practice and learn how to build websites with React.

## Why I made this

- To learn how to build a website using React
- To understand how styling works with CSS Modules
- To get better at writing clean code

## What I used

- **React** — to build the pages and components
- **React Router** — to move between pages without reloading
- **Vite** — to run the project while working on it
- **CSS Modules** — to style each part of the website

## What the website can do for now is

- Show all products in a grid
- Click on a product to see more details about it
- Add products to favorites
- Move between pages without the page reloading
